Intro to the Web

What is the Web?

The world wide web is a collection of HTML documents.

HTML = HyperText Markup Language.

The links between these pages are called hyperlinks.

Major pieces

Your browser sends requests via the internet to servers and servers respond with files. These servers are computers just like yours, except that they are optimized for sitting in a closet serving files.

HTML attributes

  • anchor tags have the href attribute. 
    													
	
	<a href="www.reddit.com">derp</a>
  • image tags (called void tags because they have no content) have src and alt tags. 
    													
	
	<img src="<url>" alt="<text>">

Inline vs Block containers.

<span> elements are inline and <div> elements are block.

The DOM

The DOM ( document object model) of a web page is a tree-like structure containing all the elements in its HTML document as its nodes.

When creating a website from a design of what it will look like, it is useful to draw a box around each visual element on the page, and then code each box as an element of the DOM.

Cascading Style Sheets

"Cascading" refers to the inheritance property that CSS has. See wikipedia.

Some recommended references: css-tricks.com MDN

The box model

Each element is placed in a box on the page, which includes (from the inside out) its content, padding, border, and margin. This can be viewed in developer tools in Chrome.

Ordinarily, the size of an element as specified in your CSS will just be be the size of the content, but the size of the padding, border, and margin will also need to be taken into account for planning layouts. It can be useful to set the box-sizing property of an element to border-box. This means that the calculation of the element size will include the border and padding (but not the margin). At present, it is necessary to use vendor prefixes:

					
-webkit-box-sizing: border-box;
-moz-box-sizing: border-box;
box-sizing: border-box;

Semantic tags and content sections

In HTML5, semantic tags have been introduced to allow developers to break up the content of a page into sections. See: MDN.

Default Style Rules

Browsers use default stylesheets to determine how to display HTML elements. You can view the default style rules for the following browsers: WebKit (Chrome and Safari), Firefox, and IE.

To help ensure your website is consistent across browsers, you can use a CSS reset such as normalize.css.
